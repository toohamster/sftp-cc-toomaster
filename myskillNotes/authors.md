# 关于作者

![Author Photo](https://avatars.githubusercontent.com/u/16458414?s=400&u=7fd7d3827bd4824339e1ee5bf098fb78725728ec&v=4)

**toohamster**

GitHub: [@toohamster](https://github.com/toohamster)

## 个人简介

toohamster 是一名热爱开源的开发者，专注于自动化开发和 DevOps 工具链建设。作为 sftp-cc 项目的创始人，他致力于通过自然语言交互简化开发者与服务器之间的文件传输流程。

## 项目背景

本书源于 [sftp-cc](https://github.com/toohamster/sftp-cc) 项目的开发实践。sftp-cc 是一个 Claude Code Skill 插件，提供通用 SFTP 上传能力，支持通过 Plugin Marketplace 安装，也支持手动安装。

在开发 sftp-cc 的过程中，作者总结了一套完整的 Claude Code Skill 开发方法论，并将其整理成书，希望帮助更多开发者掌握这项技能。

## 技术栈

- **编程语言**: Shell, PHP, JavaScript
- **DevOps**: CI/CD, GitHub Actions, Docker
- **开发工具**: Claude Code, VS Code, Git

## 开源贡献

除了 sftp-cc 项目外，作者还积极参与其他开源项目的开发和贡献。相信通过开源协作，可以让技术变得更好。
