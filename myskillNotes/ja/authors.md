## プロフィール

toohamster はオープンソース愛好家でありデベロッパーで、自動化開発と DevOps ツールチェーンの構築に注力しています。sftp-cc プロジェクトの創設者として、自然言語インタラクションを通じて開発者とサーバー間のファイル転送ワークフローの簡素化に取り組んでいます。

## プロジェクトの背景

本書は [sftp-cc](https://github.com/toohamster/sftp-cc) プロジェクトの開発経験に基づいて執筆されました。sftp-cc は Claude Code Skill プラグインで、Plugin Marketplace からのインストールと手動インストールの両方をサポートする汎用 SFTP アップロード機能を提供します。

sftp-cc の開発過程中、著者は Claude Code Skill 開発の完全な方法論をまとめ、本書に整理しました。より多くのデベロッパーがこのスキルを習得することを願っています。

## 技術スタック

- **プログラミング言語**: Shell, PHP, JavaScript
- **DevOps**: CI/CD, GitHub Actions, Docker
- **開発ツール**: Claude Code, VS Code, Git
